All you need to show Untappd data on your WooCommerce store. Cheers!

== Description ==

With this plugin you can:

Show Untappd Ratings on product page instead of WooCommerce one's.
Add a Google Map Untappd Feed to your site.
Add Untappd Ratings to Structured Data.
